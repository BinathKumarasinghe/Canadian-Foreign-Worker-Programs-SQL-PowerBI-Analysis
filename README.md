🍁 Canadian Foreign Worker Programs — End-to-End Data Analysis

📌 Overview
A complete data analysis pipeline for the Canadian Foreign Worker Programs dataset,
covering SQL database design, data cleaning, query development,
and interactive Power BI dashboard visualizations.

 🗂️ Project Structure
- 🗄️ **Database:** SQL Server — `Canadian_Foreign_Worker_Programs`
- 📋 **Tables:** Provinces & Territories, Occupations, Statistics
- 📊 **Dashboards:** Power BI with interactive filters and multi-year comparisons

✨ Key Features
- 🧹 Data cleaning with NULL value detection across all critical columns
- 📅 Yearly and quarterly employment aggregation queries
- 🔍 Identification of high-fluctuation regions and industries
- 📈 Power BI visuals: pie charts, bar charts, line charts, maps, KPI cards & gauges

💡 Insights Uncovered
- 🏙️ Ontario and British Columbia consistently lead in employment totals
- 🏥 Healthcare and IT sectors show the strongest occupational growth
- 📆 Q3 dominates quarterly employment figures (28.17% in 2015)
- 📈 Overall employment grew ~10% from 2015 to 2016
- 🎯 KPI target of 30,090 surpassed at 35,320 (+17.38%)

🛠️ Tools & Technologies
- 🖥️ Microsoft SQL Server
- 📊 Power BI Desktop
- 💾 SQL (DDL, DML, Aggregations)
